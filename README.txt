在这些作品中，他的很多品味奇特的台词，生动（生草）的表情和动作被大量当成梗，使他一跃成为了由本片发展而来的《仲夏夜之淫梦》系列中人气最高的角色。

根据野兽先辈本人在片中采访里的说法，他在当时（即1999年时）24岁，身高170cm，体重74kg，是在校学生。由于Babylon系列“素人出演”的性质，他很可能并不是一位专业㚻片演员而是临时演员，加上色情片行业严格的保密制度，他的真实身份和去向无从考证。野兽先辈有时会被称为田所浩二，这个名字来源于2011年被新闻报导的一个可能并无关系的猥亵罪嫌疑犯“田所浩治”（同音），后来因为输入法比较靠前的原因使得田所浩二沿用至今，但需要注意的是田所浩二并非先辈的本名。

野兽先辈有人间之屑（人渣）的评价，甚至被淫梦民认为种族是野兽，因此野兽先辈的同人画像中，颜值较好的甚至被打上了“拟人化”的Tag（无慈悲的淫梦民）。其野兽的目光和野兽的咆哮以及看过的次数比亲生父母还多的脸已经成为代表性的桥段，与截图构图相同的各种同人图层出不穷，影片中迫真的演技与对后辈温柔的表情也很草生（搞笑）。

注意：这里并非是对演员本人的评价，而是对他所扮演的角色“田所”的评价，请不要误会。

由于在音MAD中常常使用野兽的咆哮（叫床）音作为音源，经过加工后反而变成美声乐器，因此又有乐圣的称呼，通常在音MAD中担任电吉他的音源（イキスギター）。如今的野兽先辈已然成为素材的宝库兼名言制造机。获得称号不计其数，例如粪便的拟人化（うんこの擬人化）、只是在做爱就很好笑的男人（日文：セックスしてるだけで面白い男）、所有动作都被去背景化的男人（日文：すべての動作を切り抜かれる男）、世上高潮表情被见过最多次的男人（日文：世界で一番イキ顔を見られた男），等等。

面试篇
在拍摄㚻片的面试中称自己24岁，是学生。身高是170公分，体重是74公斤。没有特别在做什么运动，不过有在健身，身体很结实（身体不错，蛮结实的），因此有时衣服会出现尺寸不对的情况。去年和女朋友分手。曾经去过风俗店，比较喜欢王道征途，泡泡系。对撸炮很是赞赏（“压力马斯内！”），表示自己撸炮挺频繁的，上次大概是三天前。脱衣服的时候被别人看到完全不会害羞，体现出王者一般的游刃有余。喜欢晒太阳，所以晒痕很明显。
